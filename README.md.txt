# Global Sustainable Energy — Case Study

This project analyzes global sustainable-energy indicators (2000–2020) using machine learning and clustering techniques.

## Contents
- Dataset: `data/global-data-on-sustainable-energy.csv`
- Notebook: `notebooks/GDSE_code.ipynb`
- Presentation: `presentation/Project_PPT.pptx`

## Methods
- Preprocessing: StandardScaler, missing value handling
- Clustering: KMeans, PCA
- Modeling: Linear Regression, Random Forest, Gradient Boosting
- Evaluation: R², RMSE

## Results
Random Forest performed best with highest R² score.
